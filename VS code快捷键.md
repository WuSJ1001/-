# ⌨️ Visual Studio Code 常用快捷键笔记 (Windows)

## 一、 通用快捷键 (General)
*   **Ctrl+Shift+P, F1**：显示命令面板 (Command Palette)
*   **Ctrl+P**：快速打开，转到文件 (Quick Open)
*   **Ctrl+Shift+N**：新建窗口/实例
*   **Ctrl+Shift+W**：关闭窗口/实例
*   **Ctrl+,**：用户设置 (Settings)
*   **Ctrl+K Ctrl+S**：键盘快捷键设置

## 二、 基础编辑 (Basic Editing)
*   **Ctrl+X / Ctrl+C**：剪切 / 复制当前行（空选时）
*   **Alt+ ↑ / ↓**：上下移动当前行
*   **Shift+Alt + ↓ / ↑**：向上/下复制当前行
*   **Ctrl+Shift+K**：删除当前行
*   **Ctrl+Enter / Ctrl+Shift+Enter**：在下方/上方插入新行
*   **Ctrl+Shift+\**：跳转到匹配的括号
*   **Ctrl+] / [**：行缩进 / 悬挂缩进
*   **Home / End**：光标跳转至行首 / 行尾
*   **Ctrl+Home / End**：跳转至文件头 / 文件尾
*   **Ctrl+↑ / ↓**：逐行上下滚动
*   **Alt+PgUp / PgDn**：逐页上下滚动
*   **Ctrl+Shift+[ / ]**：折叠 / 展开代码块 (Fold/Unfold)
*   **Ctrl+K Ctrl+[ / ]**：折叠 / 展开所有子区域
*   **Ctrl+K Ctrl+0 / J**：折叠 / 展开所有区域
*   **Ctrl+K Ctrl+C / U**：添加 / 取消行注释
*   **Ctrl+/**：切换行注释
*   **Shift+Alt+A**：切换块注释
*   **Alt+Z**：切换自动换行 (Word Wrap)

## 三、 导航 (Navigation)
*   **Ctrl+T**：显示所有符号 (Symbols)
*   **Ctrl+G**：转到指定行
*   **Ctrl+Shift+O**：转到符号 (Symbol)
*   **Ctrl+Shift+M**：显示问题面板
*   **F8 / Shift+F8**：跳转到下一个 / 上一个错误或警告
*   **Ctrl+Shift+Tab**：切换编辑器组历史
*   **Alt+ ← / →**：后退 / 前进
*   **Ctrl+M**：切换 Tab 键移动焦点

## 四、 搜索与替换 (Search and Replace)
*   **Ctrl+F**：查找
*   **Ctrl+H**：替换
*   **F3 / Shift+F3**：查找下一个 / 上一个
*   **Alt+Enter**：选中所有匹配项
*   **Ctrl+D**：将选择添加到下一个查找到的匹配项
*   **Ctrl+K Ctrl+D**：将最后一个选择移动到下一个匹配项
*   **Alt+C / R / W**：切换区分大小写 / 正则表达式 / 全字匹配

## 五、 多光标与选择 (Multi-cursor and Selection)
*   **Alt+点击**：插入光标
*   **Ctrl+Alt+ ↑ / ↓**：在上方/下方插入光标
*   **Ctrl+U**：撤销上一次光标操作
*   **Shift+Alt+I**：在所选每行的末尾插入光标
*   **Ctrl+L**：选中当前行
*   **Ctrl+Shift+L**：选中当前选择的所有出现位置
*   **Ctrl+F2**：选中当前单词的所有出现位置
*   **Shift+Alt+ → / ←**：扩展 / 缩小选择范围
*   **Shift+Alt + 鼠标拖动**：列选择模式
*   **Ctrl+Shift+Alt + 方向键**：列选择
*   **Ctrl+Shift+Alt + PgUp/PgDn**：页级列选择

## 六、 高级语言编辑 (Rich Languages Editing)
*   **Ctrl+Space**：触发代码建议
*   **Ctrl+Shift+Space**：触发参数提示
*   **Shift+Alt+F**：格式化文档
*   **Ctrl+K Ctrl+F**：格式化选定部分
*   **F12**：转到定义
*   **Alt+F12**：速览 (Peek) 定义
*   **Ctrl+K F12**：在侧边打开定义
*   **Ctrl+.**：快速修复 (Quick Fix)
*   **Shift+F12**：显示引用
*   **F2**：重命名符号
*   **Ctrl+K Ctrl+X**：裁剪尾部空格
*   **Ctrl+K M**：更改文件语言

## 七、 编辑器管理 (Editor Management)
*   **Ctrl+F4, Ctrl+W**：关闭编辑器
*   **Ctrl+K F**：关闭文件夹
*   **Ctrl+\**：拆分编辑器
*   **Ctrl+ 1 / 2 / 3**：聚焦到第 1、2 或 3 编辑器组
*   **Ctrl+K Ctrl+ ←/→**：聚焦到上一个/下一个编辑器组
*   **Ctrl+Shift+PgUp / PgDn**：向左/右移动编辑器
*   **Ctrl+K ← / →**：移动激活的编辑器组

## 八、 文件管理 (File Management)
*   **Ctrl+N**：新建文件
*   **Ctrl+O**：打开文件
*   **Ctrl+S**：保存
*   **Ctrl+Shift+S**：另存为
*   **Ctrl+K S**：全部保存
*   **Ctrl+F4**：关闭文件
*   **Ctrl+K Ctrl+W**：关闭全部文件
*   **Ctrl+Shift+T**：重新打开已关闭的编辑器
*   **Ctrl+K Enter**：保持预览模式编辑器打开
*   **Ctrl+Tab**：打开下一个
*   **Ctrl+K P**：复制活动文件的路径
*   **Ctrl+K R**：在资源管理器中显示活动文件
*   **Ctrl+K O**：在新窗口/实例中显示活动文件

## 九、 显示 (Display)
*   **F11**：切换全屏
*   **Shift+Alt+0**：切换编辑器布局 (横向/纵向)
*   **Ctrl+ + / -**：放大 / 缩小界面
*   **Ctrl+B**：切换侧边栏显示/隐藏
*   **Ctrl+Shift+E**：显示资源管理器 / 聚焦
*   **Ctrl+Shift+F**：全局搜索
*   **Ctrl+Shift+G**：源代码管理
*   **Ctrl+Shift+D**：调试面板
*   **Ctrl+Shift+X**：扩展面板
*   **Ctrl+Shift+H**：全局替换文件
*   **Ctrl+Shift+J**：切换搜索详情
*   **Ctrl+Shift+U**：显示输出面板
*   **Ctrl+Shift+V**：打开 Markdown 预览
*   **Ctrl+K V**：在侧边打开 Markdown 预览
*   **Ctrl+K Z**：禅模式 (Zen Mode, 按两次 Esc 退出)

## 十、 调试 (Debug)
*   **F9**：切换断点
*   **F5**：开始/继续调试
*   **Shift+F5**：停止调试
*   **F11 / Shift+F11**：单步跳入 / 跳出
*   **F10**：单步跳过
*   **Ctrl+K Ctrl+I**：显示悬停提示

## 十一、 集成终端 (Integrated Terminal)
*   **Ctrl+`**：显示集成终端
*   **Ctrl+Shift+`**：创建新终端
*   **Ctrl+C**：复制选中内容
*   **Ctrl+V**：粘贴到活动终端
*   **Ctrl+↑ / ↓**：上/下滚动
*   **Shift+PgUp / PgDn**：上/下翻页滚动
*   **Ctrl+Home / End**：滚动到顶部/底部
