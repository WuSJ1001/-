## 一、UART 寄存器总表

| 地址       | 寄存器名称                         | 读/写 | 位域     | 功能说明                              |
| -------- | ----------------------------- | :-: | ------ | --------------------------------- |

| `0x8050` | clock off（时钟关闭）               | RW  | bit 15 | 写1：关闭 UART 时钟                     |
| `0x8052` | uart baud rate（波特率分频）         | RW  | [15:0] | UART 波特率分频系数（= UART时钟 ÷ 目标波特率）    |
| `0x8054` | uart rx start address（RX起始地址） | RW  | [15:0] | UART RX DMA 环形缓冲区起始地址指针           |
| `0x8056` | uart rx eadr（RX结束地址）          | RW  | [15:0] | UART RX DMA 环形缓冲区结束地址指针           |
| `0x8058` | uart rx rptrl（RX读指针低位）        | RW  | [15:0] | UART RX DMA 读指针（低字）               |
| `0x805a` | uart txsadr（TX起始地址）           | RW  | [15:0] | UART TX DMA 环形缓冲区起始地址指针           |
| `0x805c` | uart txeadr（TX结束地址）           | RW  | [15:0] | UART TX DMA 环形缓冲区结束地址指针           |
| `0x805e` | uart tx wptr（TX写指针）           | RW  | [15:0] | UART TX DMA 写指针                   |
| `0x8060` | uart control（UART控制）          | RW  | —      | 见下方详细位域说明                         |
| `0x810b` | uart state（UART状态）            |  R  | —      | 见下方详细位域说明                         |
| `0x810e` | uart txitems（TX未发送字节数）        |  R  | [15:0] | TX DMA 环形缓冲区中尚未发送的字节数             |
| `0x8110` | uart txrptr（TX读指针）            |  R  | [15:0] | TX DMA 读指针                        |
| `0x8112` | uart rx items（RX未读字节数）        |  R  | [15:0] | RX DMA 环形缓冲区中尚未被软件读取的字节数          |
| `0x8114` | uart rx rdptr（RX描述符读指针）       |  R  | [15:0] | RX DMA 读描述符指针                     |
| `0x8116` | uart rx rwptr（RX写指针）          |  R  | [15:0] | RX DMA 写指针（硬件自动维护，软件只读）           |

## 二、关键寄存器位域详解

### 2.1 `0x8060` — uart control（UART 控制寄存器）

|    位域     | 读/写 | 功能说明                                   |
| :-------: | :-: | -------------------------------------- |
|   bit 0   | RW  | **UART 使能**：写 1 开启 UART 功能，写 0 关闭      |
| bit [2:1] | RW  | **校验位配置**：0 = 无校验；2 = 奇校验；3 = 偶校验      |
|   bit 3   | RW  | **停止位配置**：0 = 1 位停止位；1 = 2 位停止位        |
|   bit 4   | RW  | **RTS/CTS 流控使能**：写 1 启用硬件流控            |
|   bit 5   | RW  | **ISO 7816 智能卡协议使能**：写 1 进入智能卡模式       |
|   bit 6   | RW  | **半双工单线 UART 使能**：写 1 切换至单线半双工模式       |
|   bit 7   | RW  | **波特率模式选择**：1 = 使用分频寄存器设定值；0 = 自动检测波特率 |


> **常用配置示例**：8 位数据位、无校验、1 位停止位、无流控 → 写入 `0x81`

### 2.2 `0x810b` — uart state（UART 状态寄存器）

| 位域 | 读/写 | 功能说明 |
|:----:|:-----:|---------|
| bit 1 | R | **TX 忙标志**：1 = TX 正在发送数据 |
| bit 2 | R | **RX FIFO 空标志**：1 = RX 缓冲区为空（无数据） |
| bit 3 | R | **RX FIFO 满标志**：1 = RX 缓冲区已满 |

## 三、GPIO 功能配置寄存器中 UART 相关值

> 寄存器地址：`0x8070`（GPIO 0–15）& `0x80d4`（GPIO 16–23）

| 配置值 | 引脚功能 |
|:------:|---------|
| `0x08` | UART TX 引脚 |
| `0x09` | UART RX 引脚 |
| `0x0a` | UART RTS 引脚 |
| `0x0b` | UART CTS 引脚 |

> **使用示例**（汇编语言）：
> ```asm
> jam gpcfg_uart_txd, core_gpio_conf+10      ; 配置 GPIO10 为 UART TX
> jam gpcfg_uart_rxd|gpcfg_pullup, core_gpio_conf+11  ; 配置 GPIO11 为 UART RX（带上拉）
> ```
> 注：使用 GPIO16–23 时，`core_gpio_conf` 需改为 `gpio_conf_hi`。

## 四、寄存器速查索引

| 功能分类 | 寄存器地址 | 说明 |
|---------|-----------|------|
| 时钟配置 | `0x8043`, `0x8050` | 时钟源选择、时钟开关 |
| 波特率 | `0x8052` | 分频系数 |
| RX 缓冲区配置 | `0x8054`, `0x8056`, `0x8058` | RX Ring Buffer 起止地址与读指针 |
| TX 缓冲区配置 | `0x805a`, `0x805c`, `0x805e` | TX Ring Buffer 起止地址与写指针 |
| UART 控制 | `0x8060` | 使能/校验/停止位/流控等 |
| UART 状态（只读） | `0x810b`, `0x810e`, `0x8110`, `0x8112`, `0x8114`, `0x8116` | TX/RX 状态与指针 |



在串口初始化时，为了避免出现问题，最好在每次初始化之前先将串口使能寄存器关闭，如果在串口使能时配置串口，串口配置会失效
# 串口初始化代码
```
my_feature_uart_init:
    // 0）先将串口使能寄存器关闭
    fetch 1,core_uart_ctrl

    set0 BIT_UART_CONTROL_ENABLE,pdata

    store 1,core_uart_ctrl

    // 1) 配 GPIO 复用

    arg gpcfg_uart_txd,temp

    setarg My_UART_TX_NUM         //tx引脚

    call gpio_config_function_int

    arg gpcfg_uart_rxd|gpcfg_pullup,temp

    setarg My_UART_RX_NUM         //rx引脚

    call gpio_config_function_int

    // 2) 关闭 UART

    fetch 1,core_uart_ctrl

    set0 BIT_UART_CONTROL_ENABLE,pdata

    store 1,core_uart_ctrl

    // 3) 打包 RX/TX ring buffer 地址到 mem_pdatatemp

    setarg mem_uart_rx_ringbuf

    store 2,mem_pdatatemp

    setarg mem_uart_rx_ringbuf+15      //需要配置ringbuf长度，这里是16字节的ringbuf

    istore 2,contw

    setarg mem_uart_tx_ringbuf

    istore 2,contw

    setarg mem_uart_tx_ringbuf+15      //同上

    istore 2,contw

    // 4) 初始化 UARTA DMA ring buffer

    fetch 8,mem_pdatatemp

    call uarta_init_dma_mem

    // 5) 时钟 + 波特率

    call uart_clock_select_main_freq_crystal      //选择晶振为主时钟

    arg 115200,temp                    //配置波特率

    call uarta_calc_baud_rate_config

    // 6) 使能 UART

    arg FUN_UART_CONTROL_ENABLE|FUN_UART_CONTROL_SETTING_RATE_BAUD,temp

    storet 1,core_uart_ctrl

    rtn
```

# 串口接收数据代码

core_uart_rxitems中存有当前未读取的数据的个数
```
my_uart_receive_bytes:

    fetch 2,core_uart_rxitems    // pdata = 当前ringbuf中字节数

    rtn blank                    // 没有数据直接返回

    store 2 , mem_rx_message_sum           // loopcnt = 要处理的字节数

my_uart_receive_bytes_loop:

    call uarta_prepare_rx        // 准备rx指针

    // 从ring中读1字节到临时位置

    arg mem_pdata,contw      // 用mem_pdata存单字节

    arg 1,loopcnt                //通过设置loopcnt来选择读取多少字节的数据

    call uart_copy_rx_bytes      // 读1字节 → mem_pdata

    call uarta_rxdone            // 更新ring读指针

    fetch 1,mem_pdata        // pdata = 刚读到的字节

    fetch 2,mem_rx_message_sum

    increase -1,pdata

    store 2,mem_rx_message_sum

    nbranch my_uart_receive_bytes_loop, blank  // 继续处理剩余字节

    rtn
```

# 串口发送数据

```
my_uart_send_bytes:
    call uarta_prepare_tx

    arg mem_tx_buf,contr             // 源：mem_tx_buf为要发送的数据的头指针

    fetch 1,mem_tx_len              // mem_tx_len为要发送的数据的字节数

    copy pdata,loopcnt               

    call uart_copy_tx_bytes

    call uarta_send

    rtn
```
